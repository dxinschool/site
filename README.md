# dxuwu

personal bio site with live widgets — built with react + vite.

## features

- **live data**: steam recently played, spotify now playing (via lanyard), discord status, hong kong weather & warnings (hko api)
- **synced lyrics**: word-by-word highlight via lrclib
- **blog + gallery**: state-based pages (no router), with image modal in gallery
- **left sidebar**: tab-style navigation (hidden on home)

## develop

```bash
npm install
npm run dev
```

## build

```bash
npm run build
```

## stack

react 18, vite 5, open-meteo/hko api, lanyard (discord presence), lrclib (lyrics), steam web api
